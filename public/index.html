Alejandrothegreat

A cheeky Discord music bot with a web dashboard. 🎶

Paste a Spotify link, a YouTube link, or just a song name — he'll find it and play it. Control everything from Discord or right here in your browser.

How it works

1
Sign in with Discord

Use the button up top. You'll see every server you share with the bot.

2
Pick a server

Open any server where the bot lives and you have the DJ role.

3
Play & control

Search, queue, skip, pause, shuffle, set volume — live now-playing with artwork.

Ways to control the music

🖥️ Web dashboard

Right here — the full control panel with a live progress bar and buttons.

⚡ Slash commands

/play, /skip, /queue and more, in Discord. See all commands →

🎛️ In-chat buttons

The bot posts a live "now playing" message with Pause / Skip / Shuffle / Stop buttons you can tap.

All slash commands

/play <song or link>Play or queue a song, playlist, or link (Spotify / YouTube / search)
/pausePause the current track
/resumeResume playback
/skipSkip to the next track
/stopStop playback and clear everything
/queueShow what's coming up
/nowplayingShow the current track with a progress bar
/volume <0-100>Set the volume
/shuffleShuffle the queue
/clearClear the queue (keeps the current song playing)
/remove <position>Remove one track from the queue
/loop <off·track·queue>Set the loop mode
/dj <everyone·managers·role>(Manage Server) Choose who's allowed to control the music
