      YouTube cookies

      YouTube now blocks bots with "Sign in to confirm you're not a bot". Pasting a cookie from a (throwaway) Google account fixes that and makes playback reliable.

      How to get it
      1. In Chrome/Edge, install the "Get cookies.txt LOCALLY" extension.
      2. Open youtube.com and make sure you're logged in (use a throwaway account you don't care about).
      3. Click the extension → Export (Netscape format) → open the file → copy everything → paste below.
      ✅ Cookie saved. Play a YouTube link to test.

      Add a bot

      Discord application

      Create one at discord.com/developers/applications. Under the Bot tab, enable Server Members Intent and Message Content Intent (needed for the dashboard and ! commands). Each bot needs its own separate Discord application/token.

      Add this exact URL under OAuth2 → Redirects in the Discord developer portal.

      Spotify application (optional)

      Create one at developer.spotify.com/dashboard. Spotify link/playlist search still works without this, just with lower rate limits.
